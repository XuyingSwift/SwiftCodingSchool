###### Objects and Variables ############
# RAM short for random access memory, that is available for your program to use. 
# You can think of RAM as series of numbered mailboxes that can each be used to hold a piece of data while the program is running. 

# Value: a single piece of data, stored in memory somewhere.
# Object: an object is region of storage usually memory that has a value and other associated properties. 
# Objects can be named or unnamed, named object is called a variable, and the name of the object is called an identifier.
